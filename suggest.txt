PR Title:
Add external editor support with CTRL+X shortcut

PR Description:
Adds external editor support to OpenHands CLI with CTRL+X keyboard shortcut.

## New Feature
Users can now press **CTRL+X** to open their preferred external editor for composing longer messages or complex input.

## How it works
- Press CTRL+X while in the input field
- Opens the editor defined by $VISUAL or $EDITOR environment variables
- Falls back to common editors (nano, vim, code) if env vars not set
- Content is loaded into the editor, user edits and saves
- Content is loaded back into the input field when editor closes
- User must press Enter/Ctrl+Enter to submit the edited content
- Works in both single-line and multi-line input modes
